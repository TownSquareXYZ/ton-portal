## ğŸš€ Quick Start

### Prerequisites

- Node.js 18+ 
- pnpm (recommended) or npm/yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/TownSquareXYZ/ton-portal
cd ton-portal

# Install dependencies
pnpm install

# Start development server
pnpm dev
```

```bash

pnpm start       # Start production server
pnpm lint        # Run ESLint
```
